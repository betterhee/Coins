# Coins

암호 화폐의 시세 정보와 관련 뉴스 기사를 확인할 수 있습니다.



## Feature

-   암호화폐의 시세 정보를 확인할 수 있습니다.
-   암호화폐의 시간/일/주단위 시세 차트와 관련 뉴스 기사를 확인할 수 있습니다.
